# How to Add Build 9 to Your App Store Version

## Current Situation
- You're viewing "iOS App Version 1.0"
- Build 6 (version 1.0.0) is currently selected
- You need to select Build 9 (version 1.0.1)

---

## Method 1: Click on the Build Table Row (Easiest)

1. **Look at the Build section** - you should see a table with:
   - BUILD column: Shows "6" with app icon
   - VERSION column: Shows "1.0.0"
   - HAS APP CLIP column: Shows "NO"

2. **Click directly on the Build row** (the row showing Build 6)
   - This should open a build selector

3. **Select Build 9** from the list
   - Look for "1.0.1 (9)" in the list
   - Click on it

4. **Save** the changes

---

## Method 2: Change Version First, Then Build

1. **Click on "iOS App Version 1.0"** at the top of the page
   - Or look for an **"Edit"** button near the version number

2. **Change version to 1.0.1**
   - There should be a version input field
   - Change "1.0" to "1.0.1"

3. **In the Build section**, click on Build 6
   - This should show available builds

4. **Select Build 9** (should show as "1.0.1 (9)")

5. **Click "Save"** at the top right

---

## Method 3: Create New Version 1.0.1

1. **In the left sidebar**, look for the **"+"** button next to "iOS App"
   - Or look for **"Add Version"** or **"Add Platform"**

2. **Click it** and select **"Add Version"** or **"iOS"**

3. **Enter version:** `1.0.1`

4. **In the Build section**, click **"Select a build"** or click in the build area

5. **Choose Build 9** from the list (1.0.1 (9))

6. **Save**

---

## If You Can't Find It

Try these:

1. **Click anywhere in the Build table row** (the row with Build 6)
2. **Look for a dropdown arrow** next to Build 6
3. **Look for "Change" or "Edit"** link in the Build section
4. **Check if there's a "+" button** in the Build section to add a build

---

## After Selecting Build 9

Once Build 9 is selected:
1. The Build table should show "9" instead of "6"
2. Version should show "1.0.1" instead of "1.0.0"
3. Click **"Save"** at the top right
4. Then you can add "What's New" text and submit!

---

**Try clicking directly on the Build 6 row in the table - that's usually the quickest way!**
